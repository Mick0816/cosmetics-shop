# LUMIÈRE Cosmetics EC Site - ユーザーガイド

このプロジェクトは、30代後半から40代前半の女性をターゲットにした、リピーター重視のコスメECサイトのプロトタイプです。Next.jsとバニラCSS（CSS Modules/Global CSS）を使用して構築されています。

## 🛠️ セットアップと実行方法

プロジェクトをローカル環境で実行するには、以下の手順に従ってください。

### 前提条件
- Node.js (v18以上推奨) がインストールされていること。

### インストール
プロジェクトのルートディレクトリで、依存関係をインストールします。

```bash
npm install
```

### 開発サーバーの起動
開発モードでサーバーを立ち上げます。

```bash
npm run dev
```
ブラウザで [http://localhost:3000](http://localhost:3000) を開くと、サイトが表示されます。

### 本番ビルドと実行
本番環境向けに最適化されたビルドを作成し、実行します。

```bash
npm run build
npm start
```

## ✨ 主な機能と使い方

### 1. ホームページ (`/`)
- **ヒーローセクション**: ブランドの世界観を表現するメインビジュアル。
- **リピーター向け機能**: ログイン済み（モック）の場合、「Welcome back」メッセージと共に、前回購入した商品の再購入リンクが表示されます。

### 2. 商品一覧 (`/products`)
- カテゴリ（ファンデーション、プライマーなど）や仕上がり（マット、ツヤ）で商品をフィルタリングできます（デスクトップ表示時）。
- 商品カードをクリックすると詳細ページへ移動します。

### 3. 商品詳細 (`/products/[id]`)
- **定期購入オプション**: 「Subscribe & Save 10%」を選択すると、定期購入割引が適用されます。配送頻度（1ヶ月、2ヶ月、3ヶ月）を選択可能です。
- **通常購入**: 「One-time purchase」を選択すると、通常価格で購入できます。

### 4. マイアカウント (`/account`)
- リピーターにとって最も重要なダッシュボードです。
- **次回の配送**: 次回の定期便の予定日と内容が表示されます。「Skip This Month」（今月はスキップ）などの操作ボタン（モック）があります。
- **注文履歴**: 過去の注文を確認し、「Buy Again」ボタンから簡単に再注文が可能です。

## 📁 ディレクトリ構成

- `src/app`: ページとルーティング（App Router）。
- `src/components`: 再利用可能なコンポーネント（Header, Footer, ProductCardなど）。
- `src/app/globals.css`: デザインシステム（色、フォント、共通スタイル）の定義。

## 🎨 デザインシステム
`src/app/globals.css` で定義されたCSS変数を使用しています。
- プライマリカラー: Taupe (`#8E8070`)
- 背景色: Cream/Off-white (`#F9F7F5`)
- フォント: 見出しに 'Playfair Display'、本文に 'Outfit' を使用。
